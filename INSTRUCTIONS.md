# How to Add Your Documents

## Folder Structure
```
portfolio/
├── assets/
│   ├── documents/          (Put PDFs and document images here)
│   │   ├── cover-letter.pdf
│   │   └── resume.pdf
│   └── certifications/     (Put certification images here)
│       ├── cert1.jpg
│       ├── cert2.jpg
│       └── cert3.jpg
├── index.html
├── styles.css
└── script.js
```

## Cover Letter
1. Save your cover letter as `cover-letter.pdf` (or `cover-letter.jpg`)
2. Place it in `assets/documents/` folder
3. Update `index.html` line 47 if using a different format

## Resume
1. Save your resume as `resume.pdf` (or `resume.jpg`)
2. Place it in `assets/documents/` folder
3. Update `index.html` line 56 if using a different format

## Certifications
1. Save your certification images as `cert1.jpg`, `cert2.jpg`, `cert3.jpg`
2. Place them in `assets/certifications/` folder
3. Update the certification names in `index.html` (lines 86, 91, 96)
4. To add more certifications, copy the `certification-item` div block

## File Names You Need:
- `assets/documents/cover-letter.pdf` (or .jpg, .png)
- `assets/documents/resume.pdf` (or .jpg, .png)
- `assets/certifications/cert1.jpg` (or .png)
- `assets/certifications/cert2.jpg` (or .png)
- `assets/certifications/cert3.jpg` (or .png)

Note: The website will show broken image icons until you add these files. This is normal!

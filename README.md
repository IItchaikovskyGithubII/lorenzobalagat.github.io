# Portfolio Website

A simple, responsive portfolio website built with HTML, CSS, and JavaScript. Perfect for showcasing your projects, skills, and getting in touch with potential employers or clients.

## Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Modern UI**: Clean and professional design with smooth animations
- **Smooth Navigation**: Easy navigation between sections with smooth scrolling
- **Project Showcase**: Display your projects with descriptions and technology tags
- **Skills Section**: Organized skills by category (Frontend, Backend, Tools)
- **Contact Section**: Multiple ways for visitors to get in touch
- **GitHub Pages Ready**: Easily deployable to GitHub Pages

## Customization

### Update Your Information

1. **index.html**: 
   - Replace "Your Name" with your actual name
   - Update the hero section tagline
   - Edit project descriptions and links
   - Update contact information and social links

2. **styles.css**: 
   - Change colors by modifying CSS variables at the top (`:root` section)
   - Adjust fonts and spacing as needed

3. **script.js**: 
   - Currently handles smooth scrolling and scroll animations
   - Add more interactivity as needed

## Deploying to GitHub Pages

### Step 1: Create a GitHub Repository
1. Go to [GitHub](https://github.com) and create a new repository
2. Name it `username.github.io` (replace `username` with your GitHub username)
3. Make it public

### Step 2: Upload Your Files
1. Initialize git in your project folder:
   ```bash
   git init
   git add .
   git commit -m "Initial portfolio commit"
   ```

2. Add the remote repository:
   ```bash
   git remote add origin https://github.com/username/username.github.io.git
   git branch -M main
   git push -u origin main
   ```

### Step 3: View Your Portfolio
- Your portfolio will be live at `https://username.github.io`
- It may take a minute or two to be available
